Procps-ng for Cygwin

Command line and full screen utilities for browsing procfs, a "pseudo" file system dynamically generated by the system to provide information about the status of processes and system info.

Package Contents

  • free - View amount of free and used memory
  • prockill - Send a signal to a process based on PID
  • pgrep - List processes based on name or other attributes
  • pkill - Send a signal to a process based on name or other attributes
  • pmap - Report memory map of a process
  • procps - Report information of processes
  • pwdx - Report current directory of a process
  • skill - Obsolete version of pgrep/pkill
  • snice - Renice a process
  • tload - Graphical representation of system load average
  • top - Dynamic real-time view of running processes
  • uptime - Display how long the system has been running
  • vmstat - Report virtual memory statistics
  • w - Report logged in users and what they are doing
  • watch - Execute a program periodically, showing output fullscreen

About Procps-ng

Procps is an old package that can be considered the Unix equivalent to the Windows Task Manager. Procps has been forked to procps-ng and all new developments can be tracked here.

Procps-ng on GitLab »

This project is a port of Procps-ng to Cygwin. Cygwin is a large collection of GNU and Open Source tools which provide functionality similar to a Linux distribution on Windows. Some items have been removed from the Cygwin package due to them not being relevant on Windows. Features may be added to take advantage of Windows services that don't exist on Linux.
